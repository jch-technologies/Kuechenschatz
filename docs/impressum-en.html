Use of Artificial Intelligence

AI-Based Recipe Recognition

The app offers an optional premium feature ("AI Scan") that uses artificial intelligence to recognize recipes from photos. This feature uses the Google Gemini API (Gemini Flash Vision), an AI service operated by Google Ireland Limited.

When using this feature, a compressed photo is sent via a Firebase Cloud Function to the Gemini API. The AI model analyzes the image and returns a structured recipe (name, ingredients, instructions).

Notice Regarding AI-Generated Content

  • Recognized recipes are automatically generated by AI and may contain errors (e.g., incorrect quantities, missing ingredients, incomplete instructions).
  • The user is obligated to independently verify AI-recognized recipes for accuracy before use.
  • The operator assumes no responsibility for the accuracy, completeness, or safety of AI-generated recipe content.

AI-Estimated Nutritional Values

The app offers the option to have nutritional values for recipes estimated by AI. This estimation is triggered exclusively by an explicit user action. Only recipe-related data (recipe name, ingredients, servings, language) is sent to the Google Gemini API – no personal data. The estimated nutritional values can optionally be automatically transferred to Apple Health (iOS) or Google Health Connect (Android).

All AI-estimated nutritional values are non-binding approximations. They are intended for general guidance only and do not replace professional nutritional advice or laboratory analysis. The operator assumes no responsibility or warranty for the accuracy, completeness, or reliability of these estimated values.

The user is obligated to independently verify AI-estimated nutritional values before use – particularly in case of allergies, intolerances, metabolic conditions, or special dietary requirements.
