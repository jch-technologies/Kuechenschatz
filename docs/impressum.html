Einsatz von Künstlicher Intelligenz

KI-gestützte Rezepterkennung

Die App bietet eine optionale Premium-Funktion („KI Scan"), die mithilfe von Künstlicher Intelligenz Rezepte aus Fotos erkennt. Hierfür wird die Google Gemini API (Gemini Flash Vision) eingesetzt, ein KI-Dienst der Google Ireland Limited.

Bei Nutzung dieser Funktion wird ein komprimiertes Foto über eine Firebase Cloud Function an die Gemini API übermittelt. Das KI-Modell analysiert das Bild und gibt ein strukturiertes Rezept (Name, Zutaten, Zubereitung) zurück.

Hinweise zur KI-generierten Inhalten

  • Die erkannten Rezepte werden automatisch von einer KI generiert und können Fehler enthalten (z. B. falsche Mengenangaben, fehlende Zutaten, unvollständige Zubereitungsschritte).
  • Der Nutzer ist verpflichtet, KI-erkannte Rezepte vor der Verwendung eigenständig auf Richtigkeit zu prüfen.
  • Der Betreiber übernimmt keine Gewähr für die Richtigkeit, Vollständigkeit oder Unbedenklichkeit KI-generierter Rezeptinhalte.

KI-geschätzte Nährwertangaben

Die App bietet die Möglichkeit, Nährwerte für Rezepte durch eine KI schätzen zu lassen. Diese Schätzung wird ausschließlich auf ausdrückliche Aktion des Nutzers ausgelöst. Dabei werden nur rezeptbezogene Daten (Rezeptname, Zutaten, Portionen, Sprache) an die Google Gemini API übermittelt – keine persönlichen Daten. Die geschätzten Nährwerte können optional automatisch an Apple Health (iOS) bzw. Google Health Connect (Android) übertragen werden.

Alle KI-geschätzten Nährwertangaben sind unverbindliche Richtwerte. Sie dienen ausschließlich der Orientierung und ersetzen keine professionelle Ernährungsberatung oder laboranalytische Bestimmung. Der Betreiber übernimmt keine Gewähr für die Richtigkeit, Vollständigkeit oder Zuverlässigkeit dieser Schätzwerte.

Der Nutzer ist verpflichtet, KI-geschätzte Nährwertangaben vor der Verwendung eigenständig zu prüfen – insbesondere bei Allergien, Unverträglichkeiten, Stoffwechselerkrankungen oder besonderen Ernährungsanforderungen.
